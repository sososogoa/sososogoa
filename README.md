### Hello!

`React 19` · `TypeScript` · `TanStack Query` · `Zustand` · `NestJS` · `Prisma`

`Next.js 16` · `TypeScript` · `Drizzle` · `Vercel AI SDK` · `Gemini 2.5 Flash`
